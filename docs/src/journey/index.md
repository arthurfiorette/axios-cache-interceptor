# User Journey

Welcome to the axios-cache-interceptor user journey guide. This section provides task-oriented guides to help you accomplish specific goals with the library.

## Getting Started

New to axios-cache-interceptor? Start here:

1. [Installation](./installation.md) - Install the library and set up your first cached instance
2. [Your First Cache](./first-cache.md) - Make your first cached request and understand the basics
3. [Checking Cache Status](./checking-cache.md) - Learn how to verify if a request was cached

## Common Tasks

Once you have the basics down, explore these common scenarios:

- [Caching POST Requests](./caching-post-requests.md) - Enable caching for mutations
- [Persistent Storage](./persistent-storage.md) - Use localStorage or sessionStorage
- [Custom Storage](./custom-storage.md) - Build your own storage adapter
- [Invalidate on Mutation](./invalidate-on-mutation.md) - Clear cache after updates

Follow this recommended path for the best learning experience:
